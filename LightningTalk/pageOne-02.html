I Built an App for NeXTSTEP or:

Using a Standard Cocoa Approach
  • Regular Expressions: Detect operands and operators in the text.
  • NSDecimalNumber: Does the arithmetic.
  • NSAttributedString: Stores metadata, NSDecimalNumber, and does syntax highlighting.
  • NSTextAttachment: Renders the solution inline with other text.
  • NSTextView: Does all the text composing and hard work for me.
  • NSUserDefaults: Stores font and color preferences.
