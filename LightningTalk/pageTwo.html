I Built an App for NeXTSTEP or:

I Learned to Love Ancient Objective-C

There are some basics that are lacking when compared to modern Xcode and modern Objective-C. The following useful things are all missing:

  • Autocompletion
  • Static Analysis
  • Property Syntax
  • Fast Enumeration (For In Loops)
  • Blocks (Closures)
  • NSRegularExpression (Introduced 10.7/iOS4)
  • NSDocument (Introduced 10.0)
  • NSToolbar (Introduced 10.0)
